# B01 – Java-Version prüfen

## Ausführung

Das Programm wurde über IntelliJ IDEA ausgeführt.

## Programm

```java
public class VersionCheck {
    public static void main(String[] args) {
        IO.println(System.getProperty("java.version"));
    }
}
```

## Konsolenausgabe

```text
25.0.1  
```

## Ergebnis

Die IDE verwendet Java 25. Die Aufgabe ist damit erfüllt.
